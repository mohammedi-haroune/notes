# Instaling
```
git clone https://github.com/mohammedi-haroune/notes.git
cd notes
pip install -r requirements.txt
```
# Serve
```
mkdocs serve
```

```
mkdocs gh-deploy
```

# Build website
```
mkdocs build
```
